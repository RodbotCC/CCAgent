# Temporal Continuity Steward package

This draft package converts the original Temporal Continuity Ledger implementation outline into a sub-agent configuration.

Files:

- `AGENTS.md` — human/agent-readable operating instructions.
- `temporal_continuity_steward.config.json` — machine-readable configuration for app/runtime use.

Suggested eventual repo path:

`LEDGERS/AGENTS/temporal_continuity_steward/`

This agent is designed to pair with the Global Ledger Steward:
- Global Ledger Steward protects durable project identity/rules.
- Temporal Continuity Steward protects current state, assumptions, risks, and handoff across time.
