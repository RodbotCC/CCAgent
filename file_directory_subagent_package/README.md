# File Directory Steward package

This draft package converts the File Directory Ledger implementation outline into a sub-agent configuration.

Files:

- `AGENTS.md` — human/agent-readable operating instructions.
- `file_directory_steward.config.json` — machine-readable configuration for app/runtime use.

Suggested eventual repo path:

`LEDGERS/AGENTS/file_directory_steward/`

This agent is designed to work with the existing ledger steward family:
- Global Ledger Steward protects project identity/rules.
- Temporal Continuity Steward protects current state and handoff.
- North Star Steward protects purpose/alignment.
- Open Problems Steward protects unresolved-work memory.
- File Directory Steward protects directory navigation, ownership, and edit boundaries.
