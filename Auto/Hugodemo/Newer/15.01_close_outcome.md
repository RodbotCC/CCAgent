# 15.01 — Close Outcome (template)

Filled when the lead moves to agreement / contract / invoice / booked.

```
lead_id:
name:
event_date:
final_quote:
deposit_paid_at:
contract_signed_at:
booked_status: yes / pending / cancelled
post_close_handoff_filed: yes / no  → see 15.07
referral_opportunity_filed: yes / no  → see 15.06
```

## Required fields

- Final accepted quote ($)
- Deposit amount + date received
- Contract date + version
- Event date confirmed
- Venue confirmation
- Dietary requirements locked
- Lifetime-value tag if known
