# 15.03 — Hold Outcome (template)

Lead is quiet but not dead. Distinct from block: no negative signal, just
nothing to act on.

```
lead_id:
name:
held_at:
hold_reason: silent / external-block / mid-pause-by-lead
expected_resume:
auto-cadence: off (default for hold)
manual-touch-allowed: yes / no
```

## Re-activation triggers

- Inbound message of any kind
- A specific calendar trigger (event date approaches, anniversary)
- A new product / menu offering that fits this lead's profile
