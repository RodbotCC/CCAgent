# Global Ledger Steward package

This draft package converts the original Global Ledger implementation outline into a sub-agent configuration.

Files:

- `AGENTS.md` — human/agent-readable operating instructions.
- `global_ledger_steward.config.json` — machine-readable configuration for app/runtime use.

Suggested eventual repo path:

`LEDGERS/AGENTS/global_ledger_steward/`

This is intentionally broad enough to remain implementation-neutral, while concrete enough to use in GitHub PRs, agent bootstraps, and internal app agent registries.
