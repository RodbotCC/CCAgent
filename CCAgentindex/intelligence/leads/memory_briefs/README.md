# Lead Memory Briefs

Compact one-row-per-lead briefs (engagement state, next move, latest activity). Written by `build_lead_memory_briefs.py`. Foundational input for action / conversation / owner-stage scripts.

**Output convention:** date-stamped markdown — `YYYY-MM-DD.md` for daily snapshots, `YYYY-MM-DD_HH-MM.md` for sub-daily.
