# Open Problems Steward package

This draft package converts the Open Problems Ledger outline plus the active `.md` / `.json` ledger into a sub-agent configuration.

Files:

- `AGENTS.md` — human/agent-readable operating instructions.
- `open_problems_steward.config.json` — machine-readable configuration for app/runtime use.

Suggested eventual repo path:

`LEDGERS/AGENTS/open_problems_steward/`

This agent is designed to work with the earlier ledger stewards:
- Global Ledger Steward protects project identity/rules.
- Temporal Continuity Steward protects current state and handoff.
- North Star Steward protects purpose/alignment.
- Open Problems Steward protects unresolved-work memory.
