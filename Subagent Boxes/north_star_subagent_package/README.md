# North Star Steward package

This draft package converts the original North Star Ledger implementation outline into a sub-agent configuration.

Files:

- `AGENTS.md` — human/agent-readable operating instructions.
- `north_star_steward.config.json` — machine-readable configuration for app/runtime use.

Suggested eventual repo path:

`LEDGERS/AGENTS/north_star_steward/`

This agent is designed to pair with the first two ledger stewards:
- Global Ledger Steward protects durable project identity/rules.
- Temporal Continuity Steward protects current state and handoff.
- North Star Steward protects purpose, alignment, tradeoffs, and anti-goal detection.
